# Collections Framework & Generics Demo
- Overview
This project demonstrates how to use Java Collections Framework and Generics to manage and optimize data.  
It includes examples with 'List', 'Set', 'Map', and a reusable generic utility class for sorting and filtering.

#Features
- Employee Class : Comparable, equals/hashCode for uniqueness
- Collections Usage
  - 'List<Employee>' : ordered records
  - 'Set<Employee>' : unique employees
  - 'Map<Integer, Employee>' : fast ID lookup
- Generic Utility Class ('DataSorter<T>')
  - Sort by natural order (ID)
  - Sort by custom comparator (Name)
  - Filter with predicates
- Stream API : Group employees by department
